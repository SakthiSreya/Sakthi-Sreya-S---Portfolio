# Sakthi Sreya S — Portfolio

A personal portfolio website built from scratch using HTML, CSS, and JavaScript.  

**Live Demo:** 🔗 [https://sakthisreya.github.io/Sakthi-Sreya-S---Portfolio](https://sakthisreya.github.io/Sakthi-Sreya-S---Portfolio)

---

## What's Inside
- Animated hero section with large-scale typography  
- Custom cursor  
- Scroll-driven animations (blur, scale, fade)  
- Skills, Projects, and Contact sections  
- Fully responsive across screen sizes  

---

## Tech Stack
- HTML5  
- CSS3  
- JavaScript
